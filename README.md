# Summary

**This project analyzes the 2023 data science job market to uncover salary trends, regional pay differences, and the most in-demand skills. The analysis found that technical skills such as Python, SQL, AWS, and Azure are strongly associated with higher salaries and better career opportunities. It also explores how the number of required skills can impact salary growth across different job roles, providing practical insights for aspiring data professionals and job seekers.**

## Steps Used to Create the Project
- Cleaned and transformed raw data using Power Query.
- Built data models and relationships using Power Pivot.
- Performed analysis using Pivot Tables.
- Created custom salary measures using DAX.
- Designed charts and dashboards to visualize findings clearly.
